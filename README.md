# 🩸 Blood Bank Management System
## 🔗 Live Demo
[https://showmiththira.github.io/Blood-bank-management-system-/](https://showmiththira.github.io/Blood-bank-management-system-/)

## ✨ Features
- 📊 Dashboard with real-time stat cards
- 🩸 Donor Registration & Records
- 🏥 Recipient / Hospital Blood Requests
- ✅ Request Fulfillment with Auto Stock Deduction
- 📦 Live Blood Stock Inventory (8 blood groups)
- ⚠️ Low Stock Alerts
- ⚙️ Manual Stock Adjustment (wastage/expiry correction)
- ⬇️ Export Donors & Recipients to CSV
- 🌙 Dark Themed UI
- 📱 Mobile Responsive

## 🛠️ Built With
- HTML, CSS, JavaScript
- LocalStorage (data persistence)
